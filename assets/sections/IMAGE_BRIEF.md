# IMAGE BRIEF

## このLPの前提

- サービス名: COMSBI
- 目的: 住宅会社、リフォーム会社、外壁塗装会社、屋根工事会社向けのLINE追客・顧客管理ツールとして見せる
- LPで一番伝えたいこと:
  - 見積後の追客を自動化できる
  - OB顧客の再掘り起こしができる
  - 営業管理の属人化を減らせる
- 最終的にしてほしい行動:
  - LINE登録は前提
  - その先の見積もり依頼、相談につなげたい

## 全体の画像トーン

- 実写ベースで、現場感があることを優先
- きれいすぎるIT企業感より、住宅・リフォーム業の営業現場感を重視
- 日本の住宅会社、工務店、リフォーム会社で違和感がない空気感
- スタッフは作業着、営業制服、オフィスカジュアルのいずれかで自然に見えること
- 住宅、外壁、屋根、現地調査、見積説明、スマホでのLINE対応などが伝わると良い

## 避けたい表現

- 海外っぽい住宅や街並み
- ITスタートアップ感の強い抽象ビジュアル
- コールセンターっぽい印象
- 過度に笑顔だけを強調した安っぽい素材感
- 文字情報が多すぎる画像
- AI生成特有の不自然な手、顔、スマホ、建物ディテール

## デザインの方向性

- 背景として使う画像は、余白があり文字を載せやすいこと
- 1枚で全情報を詰め込まず、シーンがすぐ理解できること
- LINE運用ツールなので、必要に応じてスマホ画面や管理画面が映り込んでいてよい
- ただし画像内に細かいUI文字は不要

## 推奨サイズ

- hero: 横長 1600 x 960 以上
- about: 横長 1400 x 900 以上
- case studies: 各画像 横長 1400 x 900 以上

拡張子は `svg` で置いていますが、実運用では `webp` か `jpg` でも問題ありません。
拡張子を変える場合は `index.html` 上部の画像変数を変更してください。

## 画像ごとの指示

### 1. ヒーロー背景

- 差し替え先:
  - `assets/sections/hero/hero-main.svg`
- 役割:
  - LP全体の第一印象をつくる画像
  - 「住宅・リフォーム業向けのサービス」と一瞬で伝える

- 構図のポイント:
  - 背景全面で使うため、主役は中央から右寄りに置くと使いやすい
  - 画面左側は少し情報量を抑えると、テキストと馴染みやすい

### 2. Aboutセクションのメイン画像
